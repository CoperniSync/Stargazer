# Stargazer

This repository contains the Unity implementation of the Stargazer project. For all purposes, this repository is a Unity project that uses the ChargerAstronomyEngine and ChargerAstronomyShared repositories as Unity based libraries.

This repository does not contain any logic regarding star, spatial indexing, or heat indexing calculations on its own.

For information on how to set up the dev environment, view our [Official Documentation](https://stargazer-documentation.onrender.com/docs/getting-started.html).

## Assets/Scripts

This directory holds all scripts used by the Unity project.

- **CelestialBodies** contains definitions for the front end representations of celestial bodies such as stars, constellations, local celestial objects, and messier objects.
- **Contracts** contains interfaces for communication with the engine
- **Core** contains scripts that pertains to scheduling tasks to be performed by the engine and those that delgate tasks to other sections of the front end
- **UI** contains scripts for the functionallity of the User Interface
